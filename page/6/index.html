使用Less对CSS进行预处理

Veröffentlicht am 2017-04-01

什么是less？

在Less中文网中介绍Less是一门CSS预处理语言，它拓展了CSS语言，增加了变量，Mixin，函数等特性，让CSS更容易维护和拓展。

如何安装和使用Less？

打开node.js，输入命令：$ npm install -g less即可安装Less。
如果想要将写好的Less转化为CSS，就要使用$ lessc style.less style.css
