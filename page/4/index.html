关于call、apply、bind

关于这三个方法我们先总结一下他们的异同：

  1. apply,call,bind三者都是用来改变函数的this对象的指向的
  2. apply,call,bind三者第一个参数都是this要指向的对象，也就是想制定的上下文
  3. apply,call,bind三者都可以利用后续参数传参
  4. apply,call是立即调用的，bind是返回对应函数，便于稍后调用

What is a REST API ?

REST 是一种描述网页结构与其他系统架构有何不同的方式。

网页是一种信息网络存在于网络之上，通过浏览器很容易访问。在许多方面都有别于电脑上其他系统结构。

REST是一种架构风格去设计类似网络这样的系统。统一接口的四个规则定义了REST最主要的框架结构。根据定义，REST的API遵循着这四个接口的规则。
